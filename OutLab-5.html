Problem Statement

You are going to build a new sorting method. We have already seen mergeSort() and quickSort() in class. The method you will build for this lab is called treeSort(). We have already seen that things can be inserted into a binary search tree and retrieved in sorted order. You are going to leverage this fact to build a sorting method based on BSTs.

The basic idea is this: Given an array of generic (Comparable) elements, put those elements into a BST. Then, traverse the tree in an appropriate fashion, putting the elements back into the array. When this is completed, the array should be in sorted order.

Here are the specifications for your solution. The method treeSort() must be a static method in the Sort class (just like we did for mergeSort() during class). It will take an array of some generic Comparable data type (just like mergeSort()). It will not return anything, instead the array that is passed will be modified so that it is now in sorted order (just like mergeSort()). Since you are using a BST to accomplish this, you will likely want another couple classes to represent that BST.

Assignment

  1. Make a Sort class like we developed for mergeSort() in which to put your treeSort() method.
  2. Add any other BST classes you need to support your solution.
  3. Make the treeSort() method so that it successfully acts as detailed above.
  4. Test your solution to confirm that it is sorting.
  5. Answer the following questions in a text file. Note, you are being graded on your thought process. Don't stress if you are just making an educated guess.
    1. Given an array of size n, what do you think the worst case running time of treeSort() is? How did you get this answer? What input could yield this running time?
    2. Given an array of size n, what do you think the best case running time of treeSort() is? How did you get this answer? What input would be needed to yield this running time?
    3. Can you think of any circumstances where treeSort() may make more sense than mergeSort() or quickSort()?
